자주 묻는 질문 

  • Microsoft Copilot 앱은 대화형 채팅 환경입니다. 개인용 Microsoft 365 Copilot은 Word, PowerPoint, Excel, Outlook과 같은 앱에 Copilot을 제공합니다. 
  • 현재 제공되는 기능에는 파일을 참조하여 Word 및 PowerPoint에서 각각 문서 또는 프레젠테이션을 시작하는 기능과 Word 또는 PowerPoint 캔버스의 Copilot 채팅 패널에서 바로 Designer를 사용하여 이미지를 생성하는 기능 등 여러 유형이 있습니다. Microsoft는 소비자가 Copilot을 활용하기 시작하면서 Microsoft 365 앱 전반에서 Copilot이 작동하는 방식을 계속해서 발전시키고 있습니다.
  • Copilot은 매일같이 사용하는 Microsoft 365 앱에 AI의 강력한 기능을 제공하여 사용자가 창의성을 높이고 잠재력을 실현하며 자신감 있게 추구할 수 있도록 지원합니다. Copilot을 사용하면 OneNote에서 다음 여행이나 이벤트를 손쉽게 계획하거나, Word에서 이력서와 자기소개서를 다듬거나, Outlook에서 적절한 어조로 멋진 전자 메일을 빠르게 작성하고 긴 전자 메일 스레드를 빠르게 파악하거나, 다음번 PowerPoint 발표에 대비해 슬라이드를 빠르게 만들 수 있습니다. Copilot은 사용자와 함께 작업하면서 사용자가 흐름을 유지하고 더 의미 있는 일에 시간을 할애할 수 있도록 바쁜 업무를 처리해 주는 AI 컴패니언입니다. 
  • Microsoft 365 Family와 Microsoft 365 Premium은 개인 사용자를 위해 설계되었으며, 최대 6명까지 공유할 수 있습니다. 단, AI 기능은 구독자만 이용 가능합니다. 직장 내 협업을 위해서는 팀과 조직에 최적화된 Microsoft 365 비즈니스 플랜인 Microsoft 365 Copilot, Microsoft 365 Business Basic, Standard, Premium을 권장합니다. 모든 조직을 위한 Copilot 플랜을 확인하세요.
  • Word, Excel, PowerPoint, Outlook과 같은 Microsoft 365 앱의 AI 기능에 대한 액세스와 Designer와 같은 앱에서 AI 기능을 더 많이 사용하는 기능은 Microsoft 365 Family 플랜의 구독 소유자에게만 제공되며 다른 사람과 공유할 수 없습니다.
  • Frontier 프로그램은 Microsoft 365 Copilot의 실험적 기능에 사전 액세스할 수 있도록 하며, 고객과 크리에이터가 최신 AI 혁신을 정식 출시 전에 체험할 수 있도록 설계되었습니다. 이 기능들은 아직 개발 중이며, 향후 변경될 수 있습니다. Microsoft Frontier 프로그램에 대해 자세히 알아보세요.
  • Copilot을 사용하면 화면이나 탭 간에 전환할 필요 없이 매일 사용하는 Microsoft 365 앱에서 강력한 AI 기능을 편리하게 활용할 수 있어 환경이 더욱 원활해집니다. Copilot은 Microsoft 365 앱에서 콘텐츠에 서식을 적용할 수 있으므로 많은 사람들이 거의 사용하지 않는 수백 가지 기능을 활용할 수도 있습니다. 또한 사용자는 Word, Excel, PowerPoint와 같은 앱에서 Copilot으로 생성된 콘텐츠를 실시간으로 공유하여 공동 작업할 수 있습니다.
